字符串

str.upper() lower() title()：大写 小写 首字母大写

int(str)：将字符型数字转换为整型

列表

列表：一系列按特定顺序排列的元素，如fruits = ['apple', 'banana', 'pear']

列表操作：

访问 数列索引

print(fruits)
print(fruits[0].title())
fruits[-2]

追加 .append(元素)

fruit.append(watermelon)

插入 .insert(位置，元素)

fruit.insert(0, watermelon)

删除 del & .pop(位置) & .remove(元素)

del fruit[-1]

favorite = fruit.pop()

temp = fruit.pop(2)

fruits.remove('apple')

排序 永久.sort() & 暂时sorted(表名)

#方法与函数的区别

sorted(fruits)

fruit.sort(reverse = True)

倒序 .reserve()

print(fruits.reverse())

表长 len(表名)

len(fruits)

切片

fruits[a : b] #下标为 a 到 b-1 的列表

numbers = list(range(1, 10, 2))

函数

参数

  • 位置实参 fun(par_1, par_2) 顺序不能变

  • 关键字实参 fun(par_1 = x, par_2 = y) 顺序无所谓

  • 默认参数 def fun(par_1, par_2 = 'dog') 可以缺省

  • 列表参数

    ​ fun(names) 直接改变

    ​ fun(names[:]) 传递副本，不改变原列表值

  • 任意数量实参 fun(*parametr) 创建一个空元组，可以接收任意数量的参数

  • 任意数量关键字实参 fun(**parameter) 创建一个空字典，可以接收任意数量的关键词参数

指定别名

用 as，即from math import sqrt as f，import math as M

函数编写规范

参数列表中等号两边==不要==有空格

函数间用==两个==换行分隔开来

类

  • 形参 self ==必不可少==，还必须位于其他形参的==前面==。每个与类相关联的方法调用都自动传递实参self ，它是一个指向实例本身的引用，让实例能够访问类中的属性和方法。

  • 初始化，mydog = Dog(‘doudou’, 3)，此时python使用这两个实参==自动调用==__init()__方法

  • 方法 _init()_ 并未显式地包含 return 语句， 但Python自动返回一个表示这个小狗的实例，我们将这个实例存储在声明的变量 my_dog 中

继承

如果编写的类是现有类的一个特殊版本，那么可以使用继承，而非再从空白新建一个

一个类继承另一个类时，将自动获得另一个类的所有属性和方法，同时可以再定义自己的属性和方法

  • 父类必须在当前文件中，且在子类的前面
  • 子类类名后的括号里必须指明父类的名字，如父类为class Father(): 则子类为class Son(Father):
  • 子类的方法__init()__需要包含创建父类的所有参数
