ASSERT_* #

EXPECT_EQ(x,y)  "xдёҺyдёҚзӣёзӯү"

ж–ӯиЁҖеҲҶзұ» #

еүҚзјҖйғҪдјҡжҳҜ ASSERT_жҲ–иҖ… EXPECT_,е®ғд»¬зҡ„еҢәеҲ«дёҠйқўе·Із»ҸиҝӣиЎҢдәҶиҜҙжҳҺпјҢжүҖд»Ҙд»ҘдёӢйғҪз”Ё X_жқҘз•ҘеҶҷ

еҹәжң¬ж–ӯиЁҖ #

  • X_TRUE(condition)пјҡж–ӯиЁҖ condition дёә True
  • X_FALSE(condition)пјҡж–ӯиЁҖ condition дёә False

жҷ®йҖҡжҜ”иҫғеһӢж–ӯиЁҖ #

  • X_EQ(v1,v2)пјҡ==
  • X_NE(v1,v2)пјҡ!=
  • X_LT(v1,v2)пјҡ<
  • X_LE(v1,v2)пјҡ<=
  • X_GT(v1,v2)пјҡ>
  • X_GE(v1,v2)пјҡ>=

C еӯ—з¬ҰдёІжҜ”иҫғеһӢж–ӯиЁҖ #

  • X_STREQ(s1,s2)пјҡs1==s2
  • X_STRNE(s1,s2)пјҡs1!=s2
  • X_STRCASEEQ(s1,s2)пјҡеҝҪз•ҘеӨ§е°ҸеҶҷпјҢs1==s2
  • X_STRCASENE(s1,s2)пјҡеҝҪз•ҘеӨ§е°ҸеҶҷпјҢs1!=s2 жіЁж„Ҹпјҡ
  • Null жҢҮй’Ҳе’Ңз©әеӯ—з¬Ұ"“жҳҜдёҚзӣёеҗҢзҡ„
  • еҒҮеҰӮ char *s1 = “abc”пјҢchar *s2 = “abc”пјҢйӮЈд№Ҳ X_EQ(s1,s2)дёҚйҖҡиҝҮпјҢеӣ дёә s1 дёҺ s2 е®һйҷ…дёҠжҳҜең°еқҖжҢҮй’ҲпјҢдёҚзӣёеҗҢ;X_STREQ(s1,s2)йҖҡиҝҮпјҢеӣ дёәеӯ—з¬ҰдёІзӣёеҗҢ

жө®зӮ№ж•°жҜ”иҫғеһӢж–ӯиЁҖ #

еҜ№дәҺжө®зӮ№ж•°пјҢж–ӯиЁҖеҸӘжҳҜеҲӨж–ӯеҮ д№Һзӣёзӯү

  • X_FLOAT_EQ(f1,f2)пјҡf1 е’Ң f2 дёӨдёӘ float еҖјеҮ д№Һзӣёзӯү
  • X_DOUBLE_EQ(f1,f2)пјҡf1 е’Ң f2 дёӨдёӘ double еҖјеҮ д№Һзӣёзӯү
  • X_NEAR(v1,v2,abs_error)пјҡv1 е’Ң v2 дёӨдёӘжө®зӮ№ж•°зҡ„еҖје·®зҡ„з»қеҜ№еҖјдёҚи¶…иҝҮ abs_error

жҳҺзЎ®зҡ„жҲҗеҠҹдёҺеӨұиҙҘ #

  • SUCCEED()пјҡз”ҹжҲҗдёҖдёӘжҲҗеҠҹпјҢж”ҫиЎҢпјҢдҪҶжҳҜе№¶дёҚд»ЈиЎЁж•ҙдёӘжөӢиҜ•жҲҗеҠҹ
  • FAIL()пјҡз”ҹжҲҗиҮҙе‘Ҫй”ҷиҜҜпјҢз«ӢеҚіз»ҲжӯўеҪ“еүҚжөӢиҜ•
  • ADD_FAILURE()пјҡз”ҹжҲҗйқһиҮҙе‘Ҫй”ҷиҜҜпјҢз»§з»ӯиҝҗиЎҢжөӢиҜ•
  • ADD_FAILURE_AT(“file_path”,line_number)пјҡз”ҹжҲҗйқһиҮҙе‘Ҫй”ҷиҜҜпјҢиҫ“еҮәж–Үд»¶еҗҚе’ҢиЎҢеҸ·
  • GTEST_SKIP()пјҡзӣҙжҺҘз»“жқҹеҪ“еүҚжөӢиҜ• жҳҺзЎ®зҡ„жҲҗеҠҹдёҺеӨұиҙҘзӣёиҫғдәҺеүҚйқўзҡ„ж–ӯиЁҖжӣҙйҖӮеҗҲеҲӨж–ӯжқЎд»¶еӨҚжқӮзҡ„жғ…еҶөпјҢеӣ дёәеҲӨж–ӯжқЎд»¶еӨҚжқӮдёҚйҖӮеҗҲеҶҷжҲҗдёҖдёӘиЎЁиҫҫејҸ condition з”ЁдәҺеҲӨж–ӯгҖӮдҫӢеҰӮif...else if...else if... else...

ејӮеёёж–ӯиЁҖ #

з”ЁдәҺйӘҢиҜҒдёҖж®өд»Јз ҒжҳҜеҗҰжҠӣеҮәз»ҷе®ҡзұ»еһӢзҡ„ејӮеёё

  • X_THROW(statement,exception_type)пјҡstatement д»Јз ҒдјҡжҠӣеҮә exception_type зҡ„ејӮеёё
  • X_ANY_THROW(statement)пјҡstatement д»Јз ҒдјҡжҠӣеҮәејӮеёёпјҢдёҚйҷҗејӮеёёзұ»еһӢ
  • X_NO_THROW(statement)пјҡstatement д»Јз ҒдёҚдјҡжҠӣеҮәд»»дҪ•зұ»еһӢејӮеёё

иҮӘе®ҡд№үеёғе°”еҮҪж•°ж–ӯиЁҖпјҲи°“иҜҚж–ӯиЁҖпјү #

  • X_PREDn(fun,v1,v2…)пјҡжӢҘжңү n дёӘеҸӮж•°зҡ„еҮҪж•° fun дјҡиҝ”еӣһ True дҫӢеҰӮжңүдёҖдёӘеҮҪж•° equal(a,b)пјҢйӮЈд№Ҳе°ұжҳҜ ASSERT_PRED2(equal,a,b) дёҺ ASSERT_EQгҖҒASSERT_TRUE()иҝҷдәӣж–ӯиЁҖзҡ„еҢәеҲ«еңЁдәҺиҫ“еҮәзҡ„й”ҷиҜҜдҝЎжҒҜдёҚеҗҢпјҢеҗҢж—¶е®ғзҡ„еҠҹиғҪжӣҙеҠ ејәеӨ§

и°“иҜҚж јејҸеҢ–зЁӢеәҸж–ӯиЁҖ #

жҷ®йҖҡзҡ„ж–ӯиЁҖиҫ“еҮәдҝЎжҒҜзҡ„еҶ…е®№жҳҜйў„е®ҡеҘҪзҡ„пјҢеҰӮжһңжғіиҰҒиҮӘе®ҡд№үиҫ“еҮәзҡ„еҶ…е®№пјҢеҸҜд»ҘдҪҝз”Ёи°“иҜҚж јејҸеҢ–зЁӢеәҸж–ӯиЁҖ е…·дҪ“жҺҘеҸЈдҪҝз”ЁеҸҜеҸӮиҖғпјҡ EXPECT_PRED_FORMAT дёәдәҶйҒҝе…Қж–°зҡ„ж–ӯиЁҖе®ҸзҲҶзӮёејҸеўһй•ҝпјҢGoogleTest жҸҗдҫӣдәҶеҫҲеӨҡи°“иҜҚж јејҸеҮҪж•°пјҢе®ғд»¬еҸҜд»ҘдҪҝз”Ёи°“иҜҚж–ӯиЁҖзҡ„ж–№ејҸз»„иЈ…жҲҗйңҖиҰҒзҡ„ж–ӯиЁҖпјҢдҫӢеҰӮжө®зӮ№ж•°зҡ„е°ҸдәҺзӯүдәҺ

дёҖиҲ¬иҜӯжі• #

еңЁ gMock дёӯдҪҝз”Ё EXPECT_CALL()иҝҷдёӘж–ӯиЁҖе®ҸеҺ»и®ҫзҪ®дёҖдёӘ Mock Object зҡ„йў„жңҹиЎҢдёә EXPECT_CALL(mock_object, mock_method(params))... е…¶дёӯжңүдёӨдёӘж ёеҝғеҸӮж•°пјҢз¬¬дёҖдёӘжҳҜ mock_objectпјҢз¬¬дәҢдёӘжҳҜ mock_object дёӯзҡ„ж–№жі•пјҢеҰӮжһңжңүеҸӮж•°еҗҢж—¶иҰҒжҠҠеҸӮж•°дј иҝӣеҺ»пјҢжіЁж„ҸпјҢдёҚеҗҢеҸӮж•°зҡ„ mock_method еҸҜд»Ҙи®ӨдёәжҳҜдёҚеҗҢзҡ„йў„жңҹиЎҢдёә …йғЁеҲҶеҸҜд»ҘеЎ«еҶҷеҫҲеӨҡй“ҫејҸи°ғз”Ёзҡ„йҖ»иҫ‘жқҘжҢҮе®ҡиҜҘеҜ№иұЎиҜҘж–№жі•зҡ„и°ғз”ЁиҝҗиЎҢжғ…еҶө

using ::testing::Return;
...
EXPECT_CALL(mock_object, mock_method(params)).Times(5).WillOnce(Return(100)).WillOnce(Return(150)).WillRepeatedly(Return(200));

еңЁд»ҘдёҠзҡ„ж —еӯҗдёӯпјҢдёәиҜҘеҜ№иұЎзҡ„иҜҘж–№жі•жҢҮе®ҡдәҶеӣӣдёӘйў„жңҹиЎҢдёәпјҡ йҰ–е…Ҳе®ғдјҡиў«и°ғз”Ё 5 ж¬ЎпјҢз¬¬дёҖж¬Ўиҝ”еӣһ 100пјҢз¬¬дәҢж¬Ўиҝ”еӣһ 150пјҢд№ӢеҗҺзҡ„жҜҸж¬ЎйғҪиҝ”еӣһ 200

...

йў„жңҹи°ғз”Ёзҡ„ж¬Ўж•° #

еңЁйў„жңҹиЎҢдёәйғЁеҲҶпјҢжҲ‘д»¬еҸҜд»ҘжүӢеҠЁеҶҷдёҠ Times(3)жқҘжҢҮе®ҡе®ғйңҖиҰҒиў«и°ғз”Ё 3 ж¬ЎпјҢеӨҡжҲ–е°‘йғҪдјҡеҜјиҮҙжөӢиҜ•дёҚйҖҡиҝҮгҖӮ AtLeast()жҳҜеңЁж¬Ўж•°йў„жңҹйҮҢжҜ”иҫғеёёз”Ёзҡ„дёҖдёӘж–№жі•пјҢеҰӮжһңжҳҜ Times(3)пјҢйӮЈж–№жі•еҝ…йЎ»и°ғз”Ёдё”еҸӘиғҪи°ғз”Ё 3 ж¬ЎпјҢдҪҶжҳҜеҰӮжһңжҳҜ Times(AtLeast(3))пјҢйӮЈд№Ҳе°ұжҳҜиҮіе°‘и°ғз”Ё 3 ж¬Ўзҡ„ж„ҸжҖқдәҶгҖӮ жҲ‘д»¬д№ҹеҸҜд»ҘзңҒз•Ҙ Times()пјҢжӯӨж—¶ gMock дјҡй»ҳи®Өж №жҚ®жҲ‘д»¬еҶҷзҡ„й“ҫејҸи°ғз”Ёжғ…еҶөж·»еҠ  Times()пјҢе…·дҪ“и§„еҲҷи§ҒдёӢйқўзҡ„йғЁеҲҶгҖӮ

е…ідәҺж¬Ўж•°зҡ„йў„жңҹпјҢж ёеҝғзҡ„ж–№жі•жңүдёӨдёӘпјҢеҲҶеҲ«жҳҜ WillOnce()е’Ң WillRepeatedly()пјҢеүҚиҖ…иЎЁзӨәи°ғз”ЁдёҖж¬ЎпјҢеҗҺиҖ…иЎЁзӨәйҮҚеӨҚи°ғз”ЁпјҢе®ғд»¬еҸҜд»Ҙз»„еҗҲдҪҝз”ЁпјҢдҪҝз”Ёзҡ„е…·дҪ“и§„еҲҷеҰӮдёӢпјҡ

  • еҰӮжһңжІЎжңү WillOnce е’Ң WillRepeatedly()пјҢеҲҷй»ҳи®Өж·»еҠ  Times(1)
  • еҰӮжһңжңү n дёӘ WillOnceпјҢжІЎжңү WillRepeatedly()пјҢеҲҷй»ҳи®Өж·»еҠ  Times(n)
  • еҰӮжһңжңү n дёӘ WillOnceпјҢжңүдёҖдёӘ WillRepeatedly()пјҢеҲҷй»ҳи®Өж·»еҠ  Times(AtLeast(n))пјҢиҝҷж„Ҹе‘ізқҖ WillRepeatedly еҸҜд»ҘеҢ№й…Қи°ғз”Ё 0 ж¬Ўзҡ„жғ…еҶө

йў„жңҹеҸ‘з”ҹзҡ„иЎҢдёә #

еңЁд»ҘдёҠзҡ„ж —еӯҗдёӯпјҢдёәиҜҘеҜ№иұЎзҡ„иҜҘж–№жі•жҢҮе®ҡдәҶеӣӣдёӘйў„жңҹиЎҢдёәпјҡ йҰ–е…Ҳе®ғдјҡиў«и°ғз”Ё 5 ж¬ЎпјҢз¬¬дёҖж¬Ўиҝ”еӣһ 100пјҢз¬¬дәҢж¬Ўиҝ”еӣһ 150пјҢд№ӢеҗҺзҡ„жҜҸж¬ЎйғҪиҝ”еӣһ 200 еҰӮжһңеҺ»жҺү Times(5)пјҢйӮЈе°ұжҳҜз¬¬дёҖж¬Ўиҝ”еӣһ 100пјҢз¬¬дәҢж¬Ўиҝ”еӣһ 150пјҢд№ӢеҗҺжҜҸж¬ЎйғҪиҝ”еӣһ 200пјҢи°ғз”Ёж¬Ўж•°дёҚе°‘дәҺ 2 ж¬ЎпјҲWillRepeatedly еҸҜд»Ҙи°ғз”Ё 0 ж¬Ўпјү

йў„жңҹеҸ‘з”ҹйЎәеәҸ #

й»ҳи®Өжғ…еҶөдёӢпјҢжҲ‘д»¬и®ҫе®ҡеҘҪдёҖдёӘ mock еҜ№иұЎзҡ„еӨҡдёӘйў„жңҹиЎҢдёәж—¶пјҢжҳҜдёҚе…іеҝғе®ғд»¬зҡ„еҸ‘з”ҹйЎәеәҸзҡ„гҖӮдҫӢеҰӮд»ҘдёӢд»Јз ҒдёӯпјҢе…Ҳи°ғз”Ё PenDown()жҲ–иҖ…е…Ҳи°ғз”ЁдәҶ Forward(100)йғҪжҳҜж— жүҖи°“зҡ„пјҢйғҪиғҪйҖҡиҝҮжөӢиҜ•пјҡ

gMock дёӯзҡ„йў„жңҹиЎҢдёәй»ҳи®ӨжҳҜзІҳиҝһзҡ„пјҢе®ғд»¬дјҡдёҖзӣҙдҝқжҢҒеӯҳжҙ»зҠ¶жҖҒпјҲе“ӘжҖ•е®ғжүҖи§„е®ҡзҡ„йў„жңҹиЎҢдёәе·Із»Ҹе®Ңе…Ёиў«еҢ№й…ҚиҝҮдәҶпјү дҫӢеҰӮд»ҘдёӢзҡ„жғ…еҶөеҸҜиғҪдјҡеҮәй”ҷпјҢиҝҷз§ҚеҶҷжі•дёӢеҸҜиғҪжңҖеҲқжғізҡ„жҳҜиҝ”еӣһ 50гҖҒ40гҖҒ30гҖҒ20гҖҒ10 зҡ„и°ғз”Ёеҗ„дёҖж¬ЎпјҢдҪҶжҳҜеҸ‘з”ҹи°ғз”Ёж—¶е°ұжҠҘй”ҷдәҶпјҲдҫӢеҰӮз¬¬дёҖж¬Ўи°ғз”Ёиҝ”еӣһ 10пјҢиҖҢз¬¬дәҢж¬Ўи°ғз”Ёиҝ”еӣһ 20 ж—¶пјҢйў„жңҹиҝ”еӣһ 10 зҡ„йӮЈдёӘд№ҹиҝҳеӯҳжҙ»зқҖдјҡжҠҘй”ҷ(дёҚж»Ўи¶і Once дәҶ)пјү
