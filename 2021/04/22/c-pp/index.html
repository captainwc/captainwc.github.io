第2章 开始学习c++

命名空间

using namespace std

  • namespace，命名空间，用于区分封装好的代码中的不同函数，如厂家microsoft和厂家google的产品中都有一个printout()函数，那么要区分二者，可以将printout（）分别定义在micro和google两个命名空间。这样在使用时实际上是micro::printout()和google::printout()
  • std，类、函数、变量就是c++编译器的标准组件，他们都放在名称空间std中。因此iostream中的cout实际上是std::cout、std::endl等等
  • using，一条编译指令。using namespace std表示使用cout等时不用加std：：前缀。但这会导致std中的所有名称都可用，在大型项目中是一个潜在的问题。也可以单独声明，using std::cout，这样cout就可以用了

c++源码风格

  • 每条语句占一行
  • 每个函数有一个开始和结束花括号，各占一行
  • 函数中的语句相对于花括号缩进
  • 与函数名称相关的圆括号周围没有空白

类

类是用户定义的一种数据类型。要定义类，需要描述他能够表示什么信息，和能够对数据进行什么操作。类之于对象，就像类型之于变量。也就是说，类定义描述的是数据格式及其用法，而对象则是根据数据格式规范创建的实体。

函数

函数头：double sqrt(double n)后面跟{函数体}

函数声明：double sqrt(double);//function prototype

函数调用：fl1 = sqrt(x);

零碎

  1. 有的语言里面，有返回值的叫函数（function），没返回值的叫过程（procedure）
  2. 连续赋值是允许的，从右往左
  3. main函数的返回值，是返回给操作系统了，即将操作系统看作调用程序。返回0表示这正确退出，返回其他则表示有问题。在shell或windows批处理中可以通过返回值来判断程序正确执行与否
  4. #define log(x) std::cout<< x <<std::endl;
