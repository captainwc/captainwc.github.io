C++ primer plus 学习笔记

第2章 开始学习c++

命名空间

using namespace std

  • namespace，命名空间，用于区分封装好的代码中的不同函数，如厂家microsoft和厂家google的产品中都有一个printout()函数，那么要区分二者，可以将printout（）分别定义在micro和google两个命名空间。这样在使用时实际上是micro::printout()和google::printout()
  • std，类、函数、变量就是c++编译器的标准组件，他们都放在名称空间std中。因此iostream中的cout实际上是std::cout、std::endl等等
  • using，一条编译指令。using namespace std表示使用cout等时不用加std：：前缀。但这会导致std中的所有名称都可用，在大型项目中是一个潜在的问题。也可以单独声明，using std::cout，这样cout就可以用了。当然，也可将using句放在不同位置，如函数声明前、函数中等等，控制不同生效范围。

c++源码风格

  • 每条语句占一行
  • 每个函数有一个开始和结束花括号，各占一行
  • 函数中的语句相对于花括号缩进
  • 与函数名称相关的圆括号周围没有空白

类

类是用户定义的一种数据类型。要定义类，需要描述他能够表示什么信息，和能够对数据进行什么操作。类之于对象，就像类型之于变量。也就是说，类定义描述的是数据格式及其用法，而对象则是根据数据格式规范创建的实体。

函数

函数返回类型 形参列表

函数头：double sqrt(double n)后面跟{函数体}

函数声明：double sqrt(double);//function prototype

函数调用：fl1 = sqrt(x);

零碎

  1. 有的语言里面，有返回值的叫函数（function），没返回值的叫过程（procedure）
  2. cin.get();、system("pause");都可以用来使程序结束后暂停一下
  3. 连续赋值是允许的，从右往左
  4. main函数的返回值，是返回给操作系统了，即将操作系统看作调用程序。返回0表示这正确退出，返回其他则表示有问题。在shell或windows批处理中可以通过返回值来判断程序正确执行与否
  5. 宏定义:#define 标识符 任意语句
    编译运行前是直接原封不动将对应部分替换进去的
    比如#define f(x) (x*2+1)， f(a+1)在执行时就是(a+1*2+1)，a+1直接取代x的位置，不在乎运算优先级什么的，和预期不符。因此定义时括号不要省，容易出问题。应该是#define f(x) ((x)*2+1)
    还可以#define log(x) std::cout<< x <<std::endl;等等
  6. int var,这句话提供了两条信息：需要的内存，以及内存单元的名称
  7. cout << hex << a <<endl;以16进制形式输出a的值，还有oct八进制，以及默认的dec十进制，这些也都是在std中的

第3章 处理数据

命名规则

字母数字下划线，不能数字开头，区分大小写，不能用保留字；”__“开头和’_‘+大写字母开头的被保留给编译器及其使用的资源使用；命名最好有一定含义，并保持一定的合适的风格

数据类型

类型 备注
int 整形，4B，大致，绝对值20亿以内的都行（INT_MAX = 2^31 = 2147483647 $\approx$ 21亿)
long long 长整形，8B，又名long long int
double 双精度浮点数，8B
char 字符型，1B，char c1=’a’; char str1[10]=”wo ai ni”;
bool 布尔型，1B，0表示false，其余表示true，输出整形时true为1；c要引入stdbool.h
short 整形，2B，仅在处理大型整型数组，节省内存很重要时才使用

运算符

运算符 备注
算术运算符 + - * / % ++ –
关系运算符 < > <= >= == !=
逻辑运算符 && || ！
条件运算符 A ？B ：C A为真的执行B输出结果，否则执行C
位运算符 << >> & | ^ ~

位运算符

运算符 含义 语法 效果
<< 左移 a<<x 整数a按二进制位左移x位
>> 右移 a>>x 整数a按二进制位右移x位
& 位与 a & b 整数a和b二进制位对齐，按位进行与运算（11得1，其余0）
| 位或 a | b 整数a和b二进制位对齐，按位进行或运算（00得0，其余1）
^ 位异或 a ^ b 整数a和b二进制位对齐，按位进行异或运算（相同为0）
~ 位取反 ~ a 整数a的二进制位每一位都进行取反

零碎

  1. 整形字面值，是显示书写的常量。c++中，0x开头表示16进制，19开头表示10进制，0+17开头表示8进制。如int a = 0xff; int b = 042; int c = 91
